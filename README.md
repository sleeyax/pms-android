# pms-android
Run [PimpMyStremio](https://github.com/sungshon/PimpMyStremio) natively on android. 
No [hacks](https://gist.github.com/sleeyax/e9635eb352a4fcdf94194f763d743689) required.

## Setup
```Bash
$ npm install
```

## Build
Production:
```bash
$ npm run build
```

Development: 
```bash
$ npm run build:debug
```
